# Data Analysis Projects Portfolio

## Overview

This repository contains a collection of data analysis, business intelligence, and machine learning projects developed during my learning journey. The projects cover different domains including sales analysis, supply chain analytics, customer behavior analysis, accident analysis, financial analytics, and predictive modeling.

The main objective of these projects was to apply data analysis techniques, data visualization, statistical analysis, and machine learning concepts to solve real-world business problems and generate actionable insights.

---

## Projects Included

### 1. Supply Chain Dashboard

Developed an interactive dashboard to analyze supply chain performance, inventory trends, and operational KPIs.

### 2. Sales Dashboard

Built a Power BI dashboard to monitor sales performance, revenue trends, and key business metrics.

### 3. Blinkit Dashboard

Created a business intelligence dashboard to analyze retail sales, product performance, and customer purchasing patterns.

### 4. Credit Card Transaction Analysis

Performed exploratory data analysis on transaction data to identify spending trends and customer behavior.

### 5. Road Accident Analysis

Analyzed accident datasets to uncover patterns, risk factors, and insights that support data-driven decision-making.

### 6. Housing Market Analysis

Conducted data analysis on housing datasets to explore pricing trends and market characteristics.

### 7. Nobel Laureates Analysis

Explored Nobel Prize data to identify historical trends, distributions, and notable patterns.

### 8. Olympics Analysis

Analyzed Olympic Games datasets to evaluate country performance, athlete achievements, and competition trends.

### 9. Company Sales Analysis

Performed sales analytics to measure business performance and identify growth opportunities.

### 10. Machine Learning Projects

Implemented predictive and analytical models using various machine learning techniques for real-world use cases.
